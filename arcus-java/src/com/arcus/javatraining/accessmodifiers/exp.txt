Variables are of 3 types:

1. Local variable
2. Static variable
3. Instance variable


Syntax of variable declaration:

data type variable [ = value][, variable [= value] ...] ;

A variable can be declared as following ..examples

int a, b, c; // Declares three ints, a, b, and c.
int a = 10, b = 10; // Example of initialization
byte B = 22; // initializes a byte type variable B.
double pi = 3.14159; // declares and assigns a value of PI.
char a = 'a'; // the char variable a iis initialized with value 'a'
String myString = "Arcus";


Local variable: Local Variable
A variable that is declared inside the method is called local variable.

Instance variable: 
A variable that is declared inside the class but outside the method is called instance variable 

Static variable: A variable that is declared as static is called static variable. It cannot be local.
